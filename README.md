# Flappy Fairy: a simple Flappy Bird clone game made with Java
## English
My goal with this project is to learn more about Java programming and game development. The game is inspired by the classic Flappy Bird, but with a fairy theme.

I didn’t focus on creating complex, animated artwork or elaborate sound effects, since my main focus is the game’s logic and the fairy’s flight mechanics.


## Pt-BR
Meu objetivo com este projeto é aprender mais sobre programação em Java e desenvolvimento de jogos. O jogo é inspirado no clássico Flappy Bird, mas com uma temática de fada. 

Eu não me importei em criar artes complexas e animadas, ou efeitos sonoros elaborados, pois meu foco principal é a lógica do jogo e a mecânica de voo da fada.
